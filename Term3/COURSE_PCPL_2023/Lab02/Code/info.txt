КОМАНДЫ ДЛЯ РАБОТЫ С pip
Добавление библиотеки
pip install <название библиотеки1> <название библиотеки2>
Удаление библиотеки
pip uninstall <название библиотеки> -y

Создание файла с зависимостями
Получение списка всех библиотек и их зависимостей в проекте
pip freeze > requirements.txt

Создать виртуальное окружение
python3 -m venv venv
Включить виртуальное окружение
source venv/bin/activate
Установить зависимости (если надо)
pip install -r requirements.txt
Отключить виртуальное окружение
diactivate
